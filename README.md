# Slot Machine Simulation (C++)

## 🎮 Project Description
This is a console-based slot machine simulation developed in C++.

The project simulates a simple gambling-style game where the player can spin slots and receive random outcomes based on probability.

---

## 🧠 Purpose of the Project
The main goal of this project was to practice:

- C++ programming fundamentals  
- Algorithmic thinking  
- Logic development  
- Random number generation  
- Basic simulation design  

---

## ⚙️ Features
- Random slot generation  
- Win/Loss logic system  
- Simple game loop  
- Console-based interaction  
- Basic probability-based outcomes  

---

## 💻 Technologies Used
- C++
